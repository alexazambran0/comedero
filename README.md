## Terminos de Hardware
- **CPU:** Unidad Central de Procesamiento, el "cerebro" del computador.
- **Memoria RAM:** Memoria de acceso rápido para tareas activas.

---

## Terminos de Software
- **Sistema Operativo:** Software que gestiona el hardware (ej. Windows).
- **Algoritmo:** Pasos lógicos para resolver un problema.

---

## Terminos de Redes y Seguridad
- **IP:** Dirección única que identifica a un dispositivo en red.
- **Firewall:** Bloquea accesos no autorizados.

---

## Terminos Generales de Programacion
- **Variable:** Espacio en memoria para un dato que cambia.
- **Sintaxis:** Reglas para escribir el código.
